Parsing your excel file for TV data Parser


Column Setup: Prior to parsing, ensure that your Excel file includes the following columns:

device_brand: Create or edit a column with the header device_brand to specify the TV's brand.
device_model: Create or edit a column with the header device_model to specify the TV's model number.
These two column headers are essential for successful parsing. If you fail to this, no excel file will appear

Parsing Process: When you parse the Excel file with these designated columns, an automatic download will commence in your web browser.
The downloaded file will contain the corresponding energy data for the TVs, presented as new columns alongside the TV information. Importantly,
this process will not erase any other data in your original Excel file, allowing you to retain any additional information you may have included.

Data Privacy: Your Excel data will not be stored in our database at any point during this process. To ensure alignment with the library's data, please make
sure that the "device_brand" column accurately reflects the TV's actual brand. For example, entries like "Samsung," "TCL," or "IIYAMA" in the "device_brand"
column will yield successful parsing results.
